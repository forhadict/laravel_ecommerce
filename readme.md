# Laravel Ecommerce Site

## Features:
   - Registration and Login(User and Admin)
   - Forgot Password Recovery System
   - Show All Product
   - Add to Cart Product
   - Checkout Products
   - Search Product
   - Categorized Product
   
## Tools and Technology
  - HTML
  - CSS
  - Bootstrap
  - JavaScript
  - jQuery
  - AJAX
  - Laravel
  - MySQL
